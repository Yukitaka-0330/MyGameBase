・感想
MapEditorを始めて作ってみたが、とても難しく感じた。
特にレイキャストの当たり判定のところにはものすごく悩まされた。
しかし、友人などに聞きながら完全にではないが理解できる部分も増えたので、楽しく制作に打ち込むことが出来た。
今後のゲーム制作では、まずは当たり判定を改善し、最低限使えるようにしてから、機能の拡張などを図って生きたい。
